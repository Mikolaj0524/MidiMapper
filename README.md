## 📌 About

MidiMapper connects the **Novation Launch Control 3** to your system and translates MIDI input into real-time keyboard actions.

Built for speed and simplicity, it enables workflow automation and custom shortcut mapping without DAWs or complex routing.

<br>

---

## ⚡ Features

• Real-time MIDI processing with low latency  
• Fully customizable knob and pad mapping  
• System-wide keyboard event emulation  
• Clean ImGui-based interface  
• DirectX11 rendering backend for smooth performance

## 🛠️ How to Build

### Requirements
<p>Visual Studio 2022 (MSVC v143 or newer)</p>
<p>vcpkg package manager</p>

---

### 🔧 Build Steps

<p>1. git clone https://github.com/Mikolaj0524/MidiMapper.git</p>
<p>2. cd MidiMapper</p>
<p>3. Open the <b>.slnx</b> file in Visual Studio</p>
<p>4. vcpkg integrate install</p>
<p>5. Select configuration: Release | x64</p>
<p>6. Build and run</p>
